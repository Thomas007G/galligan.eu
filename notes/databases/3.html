Creating a table

CREATE TABLE students ( id_number CHAR(9), first_name VARCHAR(20), last_name VARCHAR(30), date_of_birth DATE hometown VARCHAR(30), course CHAR(5), points INTEGER );

To Remove a Table:

DROP TABLE X; DROP TABLE IF EXISTS Y;

To Constructively Alter a Table

ALTER TABLE students ADD gender CHAR 1);

To Delete a Column/Attribute

ALTER TABLE students DROP hometown;
