Set Theory

What is a set?

A Set is a collection of things

Syntax of a set:

{"first_item", "second_item"}
  • This is called set enumeration or set display or extensional definition

More complex sets:

  • In more complex cases, we specifiy a template and rule.
  • We write this between curly braces "{}"
    • {y|y is a library book with "databases" in the title}
    • {z|z is divisible by 8, z>0, z is an integer}
      • This is known as an extensional definition
