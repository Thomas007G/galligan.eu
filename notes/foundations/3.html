Functions

A function is something that has an input and an output, otherwise, it is not a function

if f:A-->B, then A is the domain, and B is the codomain

Bijective

  • Every input has only one output

Surjective

  • Every input has at least one output

A function can have an inverse funciton

A function can have an inverse if, and only if it is a bijective function An inverse function is the opposite of the original function, returning outputs to the original inputs An eg. is an encryption/decryption function
