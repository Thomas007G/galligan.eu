Functions

  • As previously stated, there are functions and variables. We learned how to declare variables, so now we can learn how to declare functions.

Why we declare functions:

  • Functions are used to do something, ie. perform a print something to the terminal.
  • They allow our code to be shorter.
  • We don't have to hardcode things (make bad code)
  • etc.

How to declare a function

def NameOfFunction(): <some_code>
  • Note how its very english-oriented (def --> define)
  • It is generally good practice to use pascal case (ThisIsPascalCase) when defining a function.
  • The parentheses () contain any variables that are not within the function, that the function depends upon.
  • Spacing in python is extremely important. After declaring a function, one must go 4 spaces in (or press tab) so that code is within the function.
    • To call a function (run the code inside of the function), simply:
    NameOfFunction()
