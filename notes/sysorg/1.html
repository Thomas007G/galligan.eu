ASCII

Numbers in ascii are 30 + number

  • 0 = 30, 5 = 35

Standard input in samphire is ASCII

in 00 ; gets user input and places it in AL register

Intuitively, subtract 30 from the ascii to give the base10 int

To move a value to memory, move from AL register

mov AL, 04 mov [<memory register location>], AL

Stack (Push, Pop) to move between registers

push AL ; moves AL value to stack pop BL ; moves value from stack to BL
