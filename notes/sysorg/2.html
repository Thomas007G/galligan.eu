Systems Organisation

How computers work:

Computers are fundamentally a series of logic gates that perform binary calculations

These calculations are performed 1000000000's of times per second. This is caused mainly, in part due to the simplicity of the calculations performed by the logic gates in the cpu

To allow for humans to interact with these logic machines, these machines are used to emulate many systems through the use of virtual machines.

Virtual Machines

  • No physical hardware directly interacts with these virtual machines (hence; virtual)
  • Run information through many layers of abstraction through these virtual machines
  • They allow for higher levels of use, allowing for humans to interact with computers after multiple layers of these virtual machines

Software

  • Programs like C, C++, Java and python are High-Level-Languages, which allows humans to use the computer to allow other humans to interact with the computer.
  • These high-level languages interact with lower-level languages like Assembly Code
