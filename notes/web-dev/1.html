HTML Structure

  • Giving a document instructions on how it is structured (annotations)

HTML (HyperText Markup Lanugage)

  • The markup turns text documents into web pages
  • Primarily indicating its structure

Element = start tag + content + end tag

  • Except void tags, which don't have any content, example; <br>

Nested Elements

  • Elements can be nested, example;
<ul> <li> Something/li> <li>Something Else</li> </ul>

Hierarchical Structure - Tree

tree - graph that follows a hierarchical structure node - items in a tree root - node where the tree starts (no parent) leaf - nodes at end, nothing out of them (no children) parent / child - obvious ancestor - parents of parents, recursive descendant - children of children, recursive sibling - from same parent
