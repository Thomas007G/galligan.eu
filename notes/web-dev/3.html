Displaying of images

  • By default, img's are displayed inline
  • Many ways for them to be displayed ina a 'block' on a new line
  • one of the ways to do this is to nest img's in a <figure> element
  • <figcaption> --> title for an image
<figure> <figcaption>A photo of a thing</figcaption> <img src="some_photo.jpg" /> <img src="another_photo.jpg" /> </figure>

Character Sets

  • A character set is a collection of characters
    • ASCII characters set is 128 characters, mostly from the modern Latin alphabet
    • The Unicode character set is currently a little over 100,000 characters
  • A coded character set assigns a unique number to each distinct character
    • In Unicode 'A' is 65, and 'a' is 97 (decimal)
  • A charaacter encoding refers to the way the numbers are converted to bytes for storage and transmission
    • ASCII uses 7 bits for every character
    • UTF-32 uses 4 bytes for every character
    • UTF-8 uses 1 byte for all ASCII characters and 2, 3 or 4 for others
