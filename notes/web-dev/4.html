URL - Uniform Resource Locator (not important, I hope...)

scheme://user:password@host:port/pathname?query#fragment

  • Most are optional; several have defaults, eg:
    • Scheme: http
    • Port: 80

    Scheme

    • Tells browser what action to take and when
      • http (default)
        • Tells browser to send out a request using HTTP
      • https
        • Similar to http, but more secure s --> secure session layer
      • file
        • Looks on local filesystem --> file:/// (third slash is because host is ommitted)
