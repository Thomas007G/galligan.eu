Typography

Text Formatting

p { color: blue; line-height: 100%; /* space between lines relative to font-size */ letter-spacing: 1em; /* space between letters */ text-align: center; text-indent: 1em; /* the amount of space preceeding the start of a piece of text */ text-transform: uppercase; /* uppercase, lowercase */ word-spacing: 1em; /* space between each word */ text-decoration: underline; text-shadow: 0.125em 0 .0625em gray; /* horizontal distance(right), vertical distance(down), colour, blur-radius */ }

Font Properties

p { font-family: 'Comic-Sans'; font-size: 2em; font-style: italic; font-weight: bold; } /* also a shorthand */ p { font italic bold 2em 'Comic-Sans'; }

Fonts

  • Proportional (as wide as required)
  • Monospace (all have same width)
    • Serifs
      • Messy, stylistic font-style --> old advice to use this for printed documents
    • Sans-Serifs
      • Cleaner, more stable --> old advice to use this for screen

font-family specifies a Font Stack

body { font-family: Verdana, Geneva, Tahoma, sans-serif; }

Font Size

Two types: Absolute, Relative

  • Absolute
    • pt, px, inches, millimetres
  • Relative
    • %, em (used to be double inherited font), rem (relative to root's font size), vh (viewport height), vw (viewport width), wmin (smallest of width or height)

Advice for Font Size

  • Absolutes only once
  • All others are relatives
    • Try to use em more --> h1 elements only(?)
