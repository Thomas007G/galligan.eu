Box-Model

Every html elements generate at least one CSS box

display:inline;
  • CSS' display property defines a box's display type
  • Two main display types:
    • Block
      • the box starts on a new line
      • it takes up the full width available (ie. the width of its conainer) --> hence they stack vertically
    • Inline
      • the box continues on the current line
      • it takes up only as much width as necessary for its content
  • Initial CSS is inline
  • Browsers make most things block

Box-Model inside out

  • Content --> Where text and images appear
  • Padding --> Space between content and border
  • Border --> A 'frame' between padding and margin
  • Margin --> Spacing between elements

Content

p { width: 30em; /* changes size of content ONLY was before ALL of it in internet explorer*/ height: 20em /* changes size of content ONLY, shouldn't really be used, doesn't allow for dynamic */ }
  • Width and height do not change inline elements --> Will be as large/small as required --> no effect if changed
  • Percentages are relative to size of container (parent)

Padding

p { padding: 2em 1em 2.5em 1.5em; /* 2 - top, 1 - right, 2.5 - bottom, 1.5 - left */ padding: 2em 1em; /* 2 - top & bottom, 1 - left & right */ padding-top: 1em; }

Background Colours and images

  • Only affects content and padding

Background Image

p { background-image: url('./background.png'); }
  • Images are tiled --> Tesselated
body { background-repeat: horizontal; /* no-repeat */ background-position: top left; }

Border

p { border-width: 0.0625em; border-style: solid; border-color: black; border: 0.0625em solid black; border-top-color: pink; /* border-image */ border-radius:1em; border-top-right-radius: 2em 1em; /* horizontal radius, vertical radius */ }

Box-Shadow

p { box-shadow-inset: 1em /*horizontal*/ 2em /*vertical*/ gray; }

Margin

p { margin: 1em 2em 1em 2em; }
  • If two elements are vertically-aligned, the distance between will be the largest margin, not added together horizontal ones don't collapse, however
