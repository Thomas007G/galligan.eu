  • Give tables captions, for instance

<th> has a scope attribute row & col to define each respective direction, giving more information for screen readers etc.

For those of poor sight:

tr:nth-child(odd) { background-color: gray; }

Allows for table rows to be followed along more easily, as opposed to all rows being the same colour
